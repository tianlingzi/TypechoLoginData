# TypechoLoginData

TypechoLoginData是一个为Typecho 1.2.1+和PHP8设计的登录记录插件，用于记录和统计用户的登录情况。

博客地址：https://www.tianlingzi.top/archives/237/

开源地址：https://github.com/tianlingzi/TypechoLoginData

## 功能特点

- ✅ 支持记录常规登录和第三方OAuth登录（QQ、微信、GitHub、微软等）
- ✅ 按年、按月、按日期范围筛选登录记录
- ✅ 显示登录用户的UID、用户名、昵称、登录时间、登录方式和IP地址
- ✅ 提供登录方式统计，显示每种登录方式的使用次数
- ✅ 支持分页显示，每页20条记录
- ✅ 响应式设计，适配不同屏幕尺寸
- ✅ 简洁现代的管理界面

## 安装方法

1. 下载插件压缩包，解压后将文件夹重命名为`TypechoLoginData`
2. 将`TypechoLoginData`文件夹上传到Typecho博客的`usr/plugins/`目录下
3. 登录Typecho后台，进入「控制台」→「插件」
4. 在插件列表中找到「TypechoLoginData」，点击「启用」按钮
5. 插件会自动创建数据存储目录和文件

## 使用说明

1. 启用插件后，登录记录会自动开始记录
2. 进入「控制台」→「登录记录」查看登录记录
3. 使用筛选功能按年、按月或日期范围查看特定时间段的登录记录
4. 点击不同的登录方式统计项查看详细记录
5. 使用分页导航浏览更多登录记录

## 数据存储

插件将登录记录存储在目录文件夹中，无需额外数据库表。

## 注意事项

- 插件支持Typecho 1.2.1+版本
- 插件需要PHP8及以上版本支持
- 请确保`data`目录具有可写权限
